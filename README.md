# LLM Learning Repository

> Personal repository for learning Large Language Models (LLMs).

## 📚 Projects

### 📖 Hands-on Large Language Models

Learning notes and code implementations from "Hands-on Large Language Models" by Jay Alammar & Maarten Grootendorst (O'Reilly).

👉 [View book examples](./hands-on-llm/)

This book covers:
- **Part I**: Understanding Language Model Fundamentals
- **Part II**: Using Pretrained Models in Practice
- **Part III**: Training and Fine-Tuning Techniques

## 🖥️ Environment

- **OS**: macOS (Apple Silicon - M4 Max)
- **Python**: 3.11+
- **Package Manager**: Conda

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/jenslewie/llm-learning.git
cd llm-learning

# Navigate to specific project
cd hands-on-llm

# Follow the README in each project for environment setup
```
